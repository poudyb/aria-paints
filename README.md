# aria-paints

**Aria Paints** — a simple static coloring app for young learners.

## Live site

[https://poudyb.github.io/aria-paints/](https://poudyb.github.io/aria-paints/)

Hosted on [GitHub Pages](https://pages.github.com/) from the `main` branch (repository root), same setup as [arias-playground-app](https://github.com/poudyb/arias-playground-app).

## Run locally

From this directory:

```bash
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080).
